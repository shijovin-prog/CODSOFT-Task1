Overview
This project focuses on designing a mobile app signup flow that provides users with a simple, intuitive, and seamless onboarding experience. The primary objective is to reduce friction during account creation while ensuring accessibility, usability, and a visually appealing interface.

Objectives
Design a user-friendly mobile signup experience.
Create a clear and consistent interface for new users.
Improve usability through intuitive navigation and feedback.
Ensure accessibility and responsive design principles.

Key Features
Welcome/Get Started screen
Sign Up form (Name, Email, Password)
Password visibility toggle
Input validation and error messages
Social login options (Google, Apple, etc.)
OTP/Email verification screen
